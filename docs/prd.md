# NeepickNaepick MVP PRD

## Goal

Let close friends compare everyday picks, choose between each other’s options, and ask the community for outside votes when the friend group is split.

## MVP scope

1. Friend/group creation.
2. Pick media upload through presigned storage URLs.
3. Pick battle creation from uploaded media and short captions.
4. Multi-dimensional friend voting for broad pick categories.
5. Results by category and overall winner.
6. Community vote request when friends want outside judgement.

## Primary pick types

- Style picks: outfits, sneakers, accessories.
- Place picks: cafes, restaurants, travel spots, date courses.
- Product picks: gadgets, gifts, home items.
- Content picks: movies, shows, playlists, games.
- Food picks still work, but they are only one category among many.

## Non-goals for first skeleton

- Payments.
- Marketplace checkout.
- Heavy recommendation ranking.
- Public-first anonymous feed.
